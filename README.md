# Pandas Practice Tasks for Data Analysis
This repository contains a Jupyter Notebook created as part of my hands-on practice while learning data analysis using Python and Pandas  
The goal of this project is to strengthen my understanding of real-world data manipulation and analysis techniques.

# Project Overview

This notebook includes a variety of practice-based tasks focused on working with tabular data using the Pandas library.  
The tasks simulate common data analyst operations such as filtering, cleaning, transforming, and exploring datasets.

This project is intended for **learning and practice purposes**.

# Tools & Technologies Used
 1. Python  
 2. Pandas  
 3. Jupyter Notebook  

# Topics Covered
  1. Loading and inspecting datasets  
  2. Selecting and filtering rows and columns  
  3. Handling missing values  
  4. Creating new columns  
  5. Dropping and renaming columns
  6. Conditional filtering using `isin()`, logical operators  
  7. Basic data exploration techniques
